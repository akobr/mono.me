# How to generate

This SDK is generated by [Kiota](https://learn.microsoft.com/en-us/openapi/kiota/overview).

## Guide

- https://learn.microsoft.com/en-us/openapi/kiota/install?tabs=powershell
- https://learn.microsoft.com/en-us/openapi/kiota/using

### Install

```powershell
dotnet tool install --global Microsoft.OpenApi.Kiota
```

### First run

```powershell
cd src/Platform/Storyteller/Sdk/
kiota generate --openapi open.api.v0.8.json --language csharp --output src
```

### Incremental updates

```powershell
cd src/Platform/Storyteller/Sdk/
kiota update --output src
```
